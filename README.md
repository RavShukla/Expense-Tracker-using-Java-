# 💰 Expense Tracker (Java + Swing + MongoDB)

## ✨ Overview

A multi-user desktop expense tracker built with Java Swing and MongoDB. Designed using layered architecture (UI → Service → DAO → Database) to ensure clean code, scalability, and maintainability.

---

## 🚀 Features

* 🔐 User authentication (register & login)
* ➕ Add expenses
* 📋 View expenses
* 🔍 Search by category
* ❌ Delete expenses
* 📊 Total expense calculation
* 👤 User-specific data isolation
